历史内容不再具有规范性，也不构成当前工作流程，不得作为新功能、持久化格式、安全边界或
实现行为的依据。当前设计只以 [`../design-baseline.md`](../design-baseline.md) 为准；当前代码
已经实现什么仍以仓库根 [`README.md`](../../README.md)、代码和测试为准。
